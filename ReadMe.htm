ASCOM Camera Driver (C#)



This driver is a simple affair, which triggers a camera through a USB-controlled relay board from KMTronic.

It allows one to use a basic camera, with an electronic release. Images are stored on the camera. A dummy USB image is returned for effect, which is stored in documents/ASCOM/Camera/usb-icon600x600.jpg

Feel free to use or modify for other USB-Serial relay boards.
